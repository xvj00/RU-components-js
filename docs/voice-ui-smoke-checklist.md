# Voice UI Smoke Checklist

Production-ready чеклист для финальной ручной приемки.

## Маршруты

- `/agent`
- `/audio-only`
- `/minimal`

## Базовые сценарии

- Страница открывается без ошибок в консоли.
- Сессия запускается и завершается штатно.
- Переключение тем визуально применяется сразу.
- Переключение плотности влияет на контрольную панель.

## Совместимость

- `VoiceAssistantControlBar` работает без новых пропсов.
- `VoiceAssistantControlBar` работает с `theme/variant/density`.
- Стили не ломают не-voice экраны.

## Persistence и fallback

- Тема восстанавливается после перезагрузки страницы.
- Плотность восстанавливается после перезагрузки страницы.
- При недоступном `localStorage` UI использует значения по умолчанию и не падает.

## Accessibility

- Полная клавиатурная навигация по интерактивным элементам.
- Видимый `focus-visible` на кнопках и переключателях.
- Достаточный контраст текста и кнопок для каждой темы.
- Длинные русские подписи не ломают верстку на узких экранах.
