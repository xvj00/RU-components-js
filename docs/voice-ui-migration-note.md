# Voice UI Migration Note

Кратко о миграции на новый персонализируемый voice UI.

## Что изменилось

- Добавлены новые визуальные параметры для `VoiceAssistantControlBar`:
  - `theme`: `calm | focus | expressive | contrast`
  - `variant`: `glass | solid`
  - `density`: `comfortable | compact`
- В demo-страницах добавлены русские подписи и переключение персональных тем.
- Настройки темы/плотности сохраняются с безопасным fallback при ошибках `localStorage`.

## Обратная совместимость

- Старое использование `<VoiceAssistantControlBar />` продолжает работать без изменений.
- Новые пропсы опциональны и имеют безопасные значения по умолчанию.

## Рекомендуемая интеграция

- На этапе внедрения передавать только `theme`, затем постепенно добавлять `variant` и `density`.
- Для сценариев с ограничениями хранилища (private mode/политики браузера) не полагаться на persisted-настройки как на единственный источник правды.
